### Postgres DB csv data insert with CLI

#### Create .env file at root directory
PGUSER= username
PGHOST= 127.0.0.1
PGPASSWORD=password
PGDATABASE= dbname
PGPORT= 5432

#### Run from terminal
- For East door
 ```node app.js East```
- For West door
 ```node app.js West```
- For Circle door
 ```node app.js Circle```
- For B2 door
 ```node app.js B2```
